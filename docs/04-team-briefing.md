# 04 — Team Briefing

## O que é o projeto

**Borrow Fighters** é um projeto indie de jogo 2D de luta com humor técnico, personagens inspirados em linguagens de programação e mecânicas baseadas em conceitos reais de computação.

O projeto está no **dia 0**. Ainda não há equipe formal, repositório de código de produção ou arte final.

## Objetivo desta fase

Reunir pessoas interessadas em discutir:

- visão do jogo;
- escopo do protótipo;
- estilo visual;
- arquitetura técnica;
- mecânicas de combate;
- personagens;
- viabilidade de produção.

## Perfil de colaboração desejado agora

Nesta fase, o projeto precisa mais de alinhamento do que execução pesada.

Ajuda útil agora:

- revisar o Mini-GDD;
- propor mecânicas simples;
- reduzir escopo;
- sugerir personagens;
- desenhar placeholders;
- validar stack técnica;
- apontar riscos;
- transformar ideias em backlog.

## O que não estamos fazendo ainda

- Contratando equipe fixa.
- Prometendo pagamento.
- Definindo arte final.
- Criando campanha de financiamento.
- Prometendo data de lançamento.
- Construindo online multiplayer.
- Fazendo escopo gigante.

## Como participar

1. Leia `docs/00-vision.md`.
2. Leia `docs/01-mini-gdd.md`.
3. Comente sugestões via issue/discussão.
4. Prefira sugestões pequenas, testáveis e compatíveis com o protótipo 0.1.

## Princípio do projeto

Primeiro provar que o jogo existe.

Depois provar que é divertido.

Depois provar que dá para produzir.
