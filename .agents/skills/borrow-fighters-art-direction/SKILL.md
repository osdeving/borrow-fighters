# Borrow Fighters Art Direction

## Workflow

1. Read `docs/07-art-direction.md`.
2. Use `references/art-review.md` for review criteria.
3. For character proposals, check `docs/templates/character-concept.md`.
4. For mood proposals, check `docs/templates/mood-proposal.md`.
5. Prefer placeholder-testable visual direction over final art.

## Review Bias

- Legibility beats detail.
- Silhouette beats lore.
- Placeholder viability beats polish.
- Technical humor should support gameplay readability.
- References must be cited and treated as inspiration, not assets to copy.

## Output Pattern

For art/mood review, respond with:

- strongest fit with current vision;
- risk to legibility or production;
- smallest placeholder test;
- whether it should be accepted, revised, or parked.
