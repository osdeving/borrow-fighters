# Fortaleza Tech Coast Arena

## Status

Placeholder candidato derivado de `assets/references/fortaleza.png`.

## Arquivos

- `assets/references/fortaleza.png`: referencia original.
- `assets/placeholder/arena-fortaleza.png`: fonte placeholder em 960x540, renderizada pela janela runtime em 1280x720.

## Uso

Arena de rotacao do prototipo. A intencao e testar um cenario brasileiro costeiro e tecnologico com leitura de:

- laboratorio costeiro e sistemas embarcados;
- infraestrutura de energia limpa;
- oceano, sensores e redes digitais como assinatura visual;
- plataforma larga e clara para luta.

## Geracao

A fonte placeholder foi redimensionada para `960x540` e recebeu leve reducao de brilho/saturacao para preservar leitura de lutadores, hitboxes, hurtboxes e texto. No runtime atual, o renderer desenha essa fonte ocupando a janela `1280x720`.

## Limitacoes

- Ainda e placeholder e nao deve ser tratada como arte final.
- A referencia possui texto/logos e elementos especificos demais para arte final; para uma versao aprovada, redesenhar com identidade propria.
