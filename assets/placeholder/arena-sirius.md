# Sirius Arena

## Status

Placeholder candidato derivado de `assets/references/sirius.png`.

## Arquivos

- `assets/references/sirius.png`: referencia original.
- `assets/placeholder/arena-sirius.png`: versao runtime em 960x540.

## Uso

Arena inicial do prototipo. A intencao e testar um cenario brasileiro de alta tecnologia com leitura forte de:

- grande instalacao cientifica;
- linhas de luz e acelerador como assinatura visual;
- plataforma circular limpa para luta;
- top-left/top-right ainda legiveis para HUD.

## Geracao

A versao runtime foi redimensionada para `960x540` e recebeu leve reducao de brilho/saturacao para preservar leitura de lutadores, hitboxes, hurtboxes e texto.

## Limitacoes

- Ainda e placeholder e nao deve ser tratada como arte final.
- A referencia possui texto/logos e elementos especificos demais para arte final; para uma versao aprovada, redesenhar com identidade propria.
